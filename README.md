# Программа обучения frontend разработке.

Рассчитываем из того, что длительность занятия 1 академический час.
___

## Нулевое занятие

Вводное занятие

### Цели занятия

* Рассказать о программирование, чем оно полезно, почему этим стоит заниматься
* Попытаться заинтересовать обучающихся
* Используя минимальные знания, сделать что-то красивое

### План занятия
1. Вступление: программирование, с чем его едят
2. Первая программа
3. Рисуем простые фигурки
4. Добавляем движение для нарисованных элементов
___

## Первое занятие

HTML разметка

### Цели занятия

* Узнать что такое тег, какие бывают теги
* Узнать что такое атрибуты тегов, для чего они нужны
* Создать HTML разметку для страницы

### План занятия

1. Выбор текстового редактора, редактор NotePad++, другие редакторы
2. Что такое тег, примеры тегов (изучаем теги `h1` - `h6`, `div`, `span`, `p`, `img`, `input`)
3. Атрибуты тегов, для чего они нужны
4. Создаем свою первую html-страничку
5. Домашнее задание - сделать разметку html-странички как на картинке
___

## Второе занятие

CSS стили или как сделать html-страницу красивой

### Цели занятия

* Познакомиться с CSS, для чего он нужен
* Научиться подключать CSS стили к html-странице
* Познакомиться с основными CSS-свойствами: `background`, `color`, `margin`, `padding`, `width`, `height`, `border`, `display`, `position`

### План занятия

1. Что такое CSS, для чего он нужен
2. Синтаксис CSS
3. Стилизуем элементы через атрибут `style`, через тег `style` и с помощью внешнего файла (используем `background`, `color`, `margin`, `padding`, `width`, `height`, `border`)
4. Порядок выставления стиля (inline > id > class > tag)
5. Свойство display, разница между блочным и инлайновым элементом
6. Позиционирование элементов на странице.
7. Домашнее задание:
    * Создать и стилизовать html-страницу как на картинке
    * Красиво стилизовать свою html-страницу
___

## Третье занятие

Знакомство с JavaScript

### Цели занятия

* Научиться подключать js к странице
* Написать первую программу
* Познакомиться с консолью разработчика
* Узнать что такое переменная, научиться создавать переменные
* Научиться работать с функциями alert, prompt, confirm

### План занятия

1. Языки программирования, отличия, немного о javascript'е
2. Создаем js-файл, пишем первую программу на js (`console.log('Hello world!')`)
3. Подключаем скрипт к странице, видим результат
4. Знакомство с консолью разработчика, зачем она нужна
5. Синтаксис js'а, создание переменной, что такое переменная
6. Функции alert, prompt, confirm
7. Пишем программу, которая приветсвует посетителя при открытии страницы
8. Домашнее задание:
    * Дописать программу
    * Написать программу, находящую площадь треугольника по трем сторонам
___

## Четвертое занятие

Знакомство с JavaScript, часть 2

### Цели занятия

* Познакомиться с типами данных
* Познакомиться с основными операторами (+, -, ==, ===, >, <, >=, <=)
* Познакомиться с условными операторами (if-else)

### План занятия

1. Знакомимся со всеми типами данных
2. Обьекты, что это такое, обращение к свойствам обьекта
3. Основные операторы, что можно делать с переменными
4. Условный оператор, тернарный оператор `?`
5. Встроеный обьект Math, метод `Math.random()`, `Math.sqrt()`
6. Пишем программу угадай число с помощью `Math.random()`, `prompt`, `alert` и `confirm`
7. Домашнее задание:
    * Дописать программу
    * Написать программу, решающую квадратные уравнения
___

## Пятое занятие

Знакомство с JavaScript, часть 3

### Цели занятия

* Познакомиться с циклами
* Узнать что такое массивы
* Познакомиться с функциями

### План занятия

1. Циклы в программировании, для чего они нужны
2. Различные циклы в js, синтаксис
3. Массивы
4. Программа выбор наибольшего/наименьшего числа в массиве
5. Что такое функция
6. Пишем первую свою собственную функцию
7. Пишем функцию, которая находит есть ли введенное число в существующем массиве.
8. Домашнее задание: написать программу, которая подсчитывает количество букв одного типа в предложении. На вход передаем предлажение и букву, которую мы хотим посчитать, на выходе получаем число - количество использований этой буквы.
___

## Шестое занятие

DOM, манипулирование элементами

### Цели занятия

* Научиться находить элементы на странице по разным селекторам
* Научиться измененять свойства элемента
* Вспомнить CSS
* Узнать что такое setTimeout и setInterval

### План занятия

1. Что такое DOM?
2. Поиск элементов: `document.getElementById()`, `document.getElementsByClassName()`, `document.getElementsByTagName()`, `document.querySelector()`, `document.querySelectorAll()`
3. Изменение свойств элементов, свойство style у HTMLElement
4. setTimeout и setInterval
5. Пишем первую анимацию на js
6. Домашнее задание: написать программу, которая будет принимать число - количество снежинок, и показывать снегопад.
___

## Седьмое занятие

События и JS

### Цели занятия

* Научиться навещивать события на элементы (клик мыши, нажатие кнопки)
* Научиться изменять свойства элементов по какому-либо событию
* События `onfocus`, `onblur`

### План занятия

1. Метод addEventListener, для чего он нужен
2. Навешиваем события мыши на кнопку
3. События клавиатуры
4. Пишем программу изменения положения элемента по нажатию на кнопки влево/вправо/вверх/вниз
5. Столкновение элементов, как определить.
5. Домашнее задание: Написать игру крестики-нолики
___

## Восьмое занятие

Финальное занятие

### Цели занятия

* Обучающийся самостоятельно пишет финальную игру - змейку
* Получить фидбек от обучающихся

### План занятия

1. Используя полученные ранее знания почти самостоятельно пишем игру
2. "Я уже все знаю", или не совсем?
___
